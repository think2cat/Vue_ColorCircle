# Vue_ColorCircle
VUE色彩环选择器

![image](./screenshot/screenshot.png)

基于 [vue2](https://github.com/vuejs/vue)

## 使用
```
<Colorcircle @change="handelChange"/>
```
绑定响应函数，返回16进制颜色值，比如 '#FFFFFF'
